== Description ==
This plugin allows Ushahidi to send and receive text messages using Telerivet. Telerivet makes it easy to send and receive SMS through an Android device (which works in any country), Twilio phone number, a phone number from Telerivet, or even your own SMS shortcode. 

== Installation ==
1. Copy the entire /telerivet/ directory into your /plugins/ directory.
2. From the Addons tab, activate the Telerivet plugin
3. Click on the [settings] link to add your Telerivet API information
4. In order to send SMS, go to the Ushahidi SMS Settings page, select 'telerivet' as the default sending provider, and add your Telerivet phone number as Phone 1
